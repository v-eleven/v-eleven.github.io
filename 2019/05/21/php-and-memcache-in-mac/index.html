Mac 下，PHP 与 memcache 的搭配食用

2019-05-21

昨日，朋友遇到一个问题，需要在 Mac 下安装 memcache 服务，并与 PHP 搭配食用。一开始还以为很简单，只要按照流程安装就好了。不曾想，全是坑啊全是坑啊全是坑！解决不易，作此记录。

基础环境

Mac OS 10.12.2 + PHP 7.1

安装 memcached

  • 安装 memcached-1.4.34，需要 libevent-2.0.22-stable
  • 安装 libevent-2.0.22-stable，需要 openssl-1.0.x

注：libevent-2.1.x 则需要 openssl-1.1.x，详见：Build fails on OSX El Capitan 10.11.6 with openssl

安装 memcached 的 PHP 扩展

  • 安装 memcached-2.2.0，需要 libmemcached library
  • 安装 libmemcached-1.0.18
